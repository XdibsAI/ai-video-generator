# 🗝️ Cara Mendapatkan OpenRouter API Key

## Langkah 1: Daftar di OpenRouter
1. Kunjungi https://openrouter.ai/
2. Klik "Sign Up" dan buat akun
3. Verifikasi email Anda

## Langkah 2: Dapatkan API Key
1. Login ke https://openrouter.ai/
2. Pergi ke https://openrouter.ai/keys
3. Klik "Create Key"
4. Beri nama key (contoh: "AI Video Generator")
5. Salin API key yang dihasilkan

## Langkah 3: Setup di Aplikasi
1. Edit file `.env`:
```bash
OPENROUTER_API_KEY=your_actual_api_key_here
